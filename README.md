# dicodigital-web
A webapp that consume dicodigital-api.

## Run the dev server

### Prerequisites

- node 4.x
- npm 3.x

```
git clone https://github.com/enguerran/dicodigital-web.git
cd dicodigital-web
npm i
```

### Build and start the dev server

```
npm start
```
